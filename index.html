Comment utiliser :

  1. Ouvrez l'application d'enregistrement sur votre montre Bangle.js
  2. Cliquez sur "Connecter à Bangle.js" ci-dessus
  3. Sélectionnez votre montre dans le menu popup
  4. Sur votre montre, sélectionnez "Send via BT"
  5. Attendez que le transfert soit terminé
  6. Cliquez sur "Sauvegarder les données" pour télécharger votre fichier CSV
